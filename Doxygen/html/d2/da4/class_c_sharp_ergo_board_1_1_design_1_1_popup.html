Detailed Description

This is intended to be a message box that is able to enter dark mode.

As regular message boxes are unable to change apperence this is made to allow for appearance changes in the message box. This is generally created to let the user know that they have made a error somewhere and need to edit or change something. The popup also produces a windows based noise for error popups.
