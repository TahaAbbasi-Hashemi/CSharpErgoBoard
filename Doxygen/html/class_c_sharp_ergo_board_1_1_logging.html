Detailed Description

The logging class is used to record what happens during runtime of the program.

This is a singleton class used to record actions done during runtime of the the program. Because the logging system is singleton based any class can call it as long as it is in the scope. This allows for benifits of not having multiple logging sections in the program. The logging class is thread safe, and uses locks.

For more information regarding singleton please see Singleton class.

◆ LogFormat

String CSharpErgoBoard.Logging.LogFormat
staticgetset

This is the format that the log will be saved as.

The default method is "%D (%T), \"F" L : M" as it contains all the nessary criteria. If you wish to create your own format you can use these variables to indicate properties of the log.

  • D
    would represent the date the log was made in YYYY-MM-DD
  • T
    would represent the time that the log was made in HH:MM:SS TT
  • t
    woudl represent the thread that called the log.
  • F
    would represent the file that the log was made in.
  • L
    would represent the line that the log was written on.
  • M
    would represent the message the log contains.
  • m
    would represent the member of the class that called the log.
