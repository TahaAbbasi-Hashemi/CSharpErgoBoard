Detailed Description

This is intended to be a extension of the combobox class provided by windows forms.

This class exists because the current combo box has a white outline surrounding the button. by overriding the draw class we are able to produce a black border ontop of the white border resolving the issue at hand. This also allows for ease of converting to dark mode.
